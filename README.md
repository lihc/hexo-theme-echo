# hexo-theme-echo

使用 main 稳定分支：

```sh
git clone -b main https://github.com/lihc/hexo-theme-echo.git themes/echo
```

使用 dev 开发分支：

```sh
git clone -b dev https://github.com/lihc/hexo-theme-echo.git themes/echo
```
